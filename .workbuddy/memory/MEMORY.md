# LiteWikiagent 项目记忆

## 品牌系统

- 品牌色：`#0D5C41` (Forest Primary) / `#00D97E` (Compile Green) / `#0A0A0A` (Ink Black) / `#F9FAFB` (Off White)
- 字体：Geist (UI) / Geist Mono (Code·Labels)
- Logo：W 形编译路径 icon mark + "LiteWiki**agent**" wordmark，agent 用品牌绿色突出
- 品牌资产：`image/` 目录下包含 logo-dark.svg、logo-light.svg、4 个 icon 变体、brand-colors.svg、BRAND.md

## README 规范

- 两个版本：`README.md`（英文）、`README_zh.md`（中文），结构一致
- 设计模式：居中 Logo（`<picture>` 暗/亮自适应）→ 徽章行 → 快速链接导航 → 折叠目录 → 内容
- 品牌徽章使用 `0D5C41` 做底色、`00D97E` 做高亮
- GitHub 引用块风格：`> [!NOTE]` / `> [!WARNING]` 用于关键提醒
- 参数使用表格呈现，代码示例嵌入 Markdown 代码块
- 内容覆盖：Philosophy → Quick Start → Core Concepts → CLI Reference → Ingest Policies → Project Structure → Development → License
